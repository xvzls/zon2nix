# zon2nix

Zon2Nix fork for converting `build.zig.zon` to a Nix
expression with all the metadata fields.

## Usage

```bash
zon2nix > build.zig.zon.nix
zon2nix zls > build.zig.zon.nix
zon2nix zls/build.zig.zon > build.zig.zon.nix
```

To use the generated file, add this to your Nix expression:

```nix
zon = callPackage ./build.zig.zon.nix {};
postPatch = ''
  ln -s ${zon.dependencies} $ZIG_GLOBAL_CACHE_DIR/p
'';
```
